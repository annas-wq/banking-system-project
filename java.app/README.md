# Java Application

This folder contains the Java application for working with PostgreSQL and MongoDB.

## Features
- Create, read, update and delete (CRUD) operations
- Support for two databases: PostgreSQL and MongoDB
- Console interface for selecting DB and CRUD actions

## How to run
1. Open the project in IntelliJ IDEA.
2. Run the `Main` class.

## Technologies
- Java 17
- Gradle
- MongoDB Driver
- PostgreSQL JDBC Driver
